# Todo Application

A full-stack Todo application built with Next.js (frontend) and FastAPI (backend).

## Project Structure

```
Todo-app/
├── .spec-kit/           # Spec-Kit configuration
│   └── config.yaml
├── specs/              # Spec-Kit managed specifications
│   ├── database/       # Database specifications
│   │   └── schema.md
│   └── ui/             # UI specifications
│       ├── components.md
│       └── pages.md
├── CLAUDE.md           # Root Claude Code instructions
├── frontend/
│   ├── CLAUDE.md
│   └── ... (Next.js app)
├── backend/
│   ├── CLAUDE.md
│   └── ... (FastAPI app)
└── README.md
```

## Features

- Add, edit, and delete todos
- Mark todos as complete/incomplete
- Filter todos (All, Active, Completed)
- User authentication (planned)
- Responsive design

## Technologies

- **Frontend**: Next.js, React, TypeScript
- **Backend**: FastAPI, Python, SQLModel
- **Database**: PostgreSQL (planned)
- **Styling**: Tailwind CSS

## Setup

1. Navigate to the frontend directory to work on the UI
2. Navigate to the backend directory to work on the API
3. Refer to the specifications in the `specs/` directory for detailed requirements"# Todo_app"
