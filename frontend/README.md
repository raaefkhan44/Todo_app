# Todo App - Professional Dark Theme with Red Accents

A modern, professional todo application with a dramatic red-black radial gradient background and sophisticated dark UI elements.

## Features

- **Professional Dark UI**: Sleek dark interface with red accents throughout
- **Dramatic Background**: Stunning radial gradient background (red to black)
- **Modern Components**: Cards with backdrop blur effects and red borders
- **Interactive Elements**: Animated buttons with gradient effects
- **Responsive Design**: Fully responsive for all device sizes
- **Theme Support**: Multiple theme options available
- **Task Management**: Complete CRUD functionality for tasks
- **Confetti Effects**: Red confetti celebration on task completion
- **Toast Notifications**: Professional notification system

## Design Highlights

- **Radial Gradient Background**: `radial-gradient(circle 1588px at -27.3% 144%, rgba(255, 22, 22, 1) 0%, rgba(0, 0, 0, 1) 43.4%, rgba(0, 0, 0, 1) 65.8%, rgba(255, 22, 22, 1) 100.2%)`
- **Button Styles**:
  - `.btn-grad`: Standard gradient button
  - `.btn-outlined`: Outlined variant
  - `.btn-grad-sm`: Small variant
  - `.btn-grad-full`: Full-width variant
- **Card Styling**: Semi-transparent dark cards with backdrop blur
- **Color Scheme**: Red accents with white text on dark backgrounds
- **Animations**: Smooth transitions and hover effects

## Components

- Landing Page with hero section and features
- Authentication pages (Sign Up/Sign In)
- Dashboard with sidebar navigation
- Task management with CRUD operations
- Settings page with theme selection
- Toast notifications
- Responsive mobile navigation

## Running the Application

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technology Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Heroicons (icons)
- Canvas Confetti (visual effects)
