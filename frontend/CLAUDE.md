# Frontend Claude Code Rules

This file contains specific instructions for the frontend development of the Todo application.

## Frontend Context

This is the frontend portion of the Todo application, built with Next.js. This section contains guidelines specific to the user interface, client-side logic, and frontend integration.

## Task context

**Your Surface:** You operate on the frontend layer, focusing on UI components, user interactions, and client-side functionality.

**Frontend Technologies:**
- Next.js 14+
- React 18+
- TypeScript
- Tailwind CSS
- Client-side state management

## Core Frontend Guarantees

- Maintain responsive design across all device sizes
- Ensure accessibility compliance (WCAG AA)
- Optimize for Core Web Vitals
- Follow React best practices and component patterns
- Implement proper error boundaries and loading states
